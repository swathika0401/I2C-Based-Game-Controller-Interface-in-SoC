# I2C-Based Game Controller Interface in SoC
Verilog HDL implementation of an I2C-based Game Controller Interface in System-on-Chip architecture with FSM control, APB integration, simulation and synthesis using Xilinx Vivado.
## Overview
This project implements an I2C-Based Game Controller Interface within a System-on-Chip (SoC) architecture using Verilog HDL. The design enables communication between a game controller and the SoC through the I2C protocol for efficient button-state data transfer.
## Features
- I2C Master Controller Implementation
- FSM-Based Communication Logic
- APB Interface Integration
- Button Status Monitoring
- Functional Verification and Waveform Analysis
- Simulation and Synthesis using Xilinx Vivado
## Architecture
The system consists of:
- I2C Master Module
- I2C Game Controller Slave Module
- APB Interface
- Finite State Machine (FSM)
- Status Registers
## Applications
- Embedded Systems
- Gaming Interfaces
- SoC Peripheral Communication
- FPGA-Based Digital System Design
## Results
The design was successfully simulated and synthesized in Xilinx Vivado. Communication between the I2C Master and Controller Slave was verified through waveform analysis, demonstrating reliable button-state transfer and protocol-compliant operation.
